Electron Community Hub

Welcome to our full-stack final project! This website allows users to browse community-created Electron plugins, monitor reported issues, and learn more about the project through a dedicated FAQ page.

The application includes a custom backend server, SQLite database, and three fully implemented features displayed below.

Browse Plugin Library → View Issue Dashboard View FAQ

Project Overview

This project includes:

  • ✓ Node.js + Express backend server
  • ✓ SQLite database with multiple tables
  • ✓ REST API endpoints for all features
  • ✓ Three interactive feature pages + FAQ

Project Features

Feature 1 – Plugin Library

Displays a list of community-created Electron plugins including their name, author, version, and rating. Users can filter plugins and add new ones to the database.

Go to Plugin Library →

Feature 2 – Issue Dashboard

Shows reported bugs and crash issues with filters for severity and status. Users can add new issues and view them in a table.

Go to Issue Dashboard →

Feature 3 – FAQ Page

Provides users with answers to common questions about the project, including how the backend, database, and features work. Includes interactive expand/collapse elements.

View FAQ →
